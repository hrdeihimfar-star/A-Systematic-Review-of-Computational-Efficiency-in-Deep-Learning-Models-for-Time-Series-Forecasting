# A-Systematic-Review-of-Computational-Efficiency-in-Deep-Learning-Models-for-Time-Series-Forecasting
This PRISMA-P systematic review examines computational efficiency in deep learning for time-series forecasting, analyzing ~2641 papers (2017–2024) from major databases. It uses AI-assisted screening and extracts model, metric, and efficiency data, assesses cost, energy, reproducibility reporting, evaluates accuracy–efficiency trade-offs and trends.
